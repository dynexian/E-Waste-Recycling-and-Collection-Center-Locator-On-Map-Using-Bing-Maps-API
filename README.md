# ♻️ E-Waste Management Locator

A web-based application that helps users locate nearby **authorized e-waste disposal centers** using the **Bing Maps API**. This project promotes sustainable practices by making it easier for people to find and safely dispose of their electronic waste.

## 📌 Features

- 🔍 **Location-based search** for nearest e-waste centers  
- 📍 **Interactive Bing Map** with live markers  
- 📏 **Real-time distance & directions** from user location  
- 📱 **Responsive design** for mobile and desktop  
- ✅ Simple, clean UI for better user experience  

---

## 🚀 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **API:** Bing Maps API  
- **Other Tools:** VS Code, Git, Netlify/Vercel (optional for deployment)
